Правила для Koala Clash с списком российских сервисов. Позволяет популярным российским сайтам и сервисам работать без VPN для повышения скорости и стабильности

# 🐨 Koala Clash RU Bypass Rules

Готовая конфигурация правила для **Koala Clash**, которая направляет трафик популярных российских сервисов напрямую (DIRECT), минуя VPN

Подходит для пользователей, которые используют VPN для зарубежных сайтов, но хотят сохранить максимальную скорость и стабильность для российских ресурсов

## ⚙️ Установка

1. Скачайте Koala Clash версии **v0.2.10** *(на новых версиях работать не будет)*
2. Откройте **Koala Clash**
3. Нажмите на кнопку **Правила**
4. Нажмите на **Карандаш** справа сверху
5. Далее нажимаем **Дополнительно**
6. В открывшемся окне редактируем правила на те, что я предоставил, для этого нажмите сначала скопируйте правила с **txt** файла и вставьте их в открывшееся окно
7. Нажимаем **Сохранить**
8. Перезапустите соединение
